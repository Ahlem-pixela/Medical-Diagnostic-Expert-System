# Medical Diagnostic Expert System

## Overview

This project is an **Expert System for Medical Diagnosis** developed in Java.  
The goal of this system is to simulate the reasoning process of a human medical expert by using a knowledge base composed of facts and rules provided by a medical specialist.

The system analyzes patient symptoms and applies predefined medical rules to suggest possible diagnoses. It demonstrates the use of **Artificial Intelligence concepts**, particularly **rule-based expert systems and inference engines**.

## Features

- User-friendly graphical interface developed with Java.
- Medical diagnosis assistance based on symptoms entered by the user.
- Knowledge representation using:
  - **Fact Base (Base des faits)**: contains medical information and symptoms.
  - **Rule Base (Base des règles)**: contains expert medical reasoning rules.
- Rule-based inference mechanism to generate diagnoses.
- Separation between knowledge base and inference process.
- Developed using Java and NetBeans IDE.
